# MyCaclulator
Normal Calculator created by using HTML and CSS

#A simple, clean calculator built with **HTML** and **CSS** 
![Image ](Image%20/Calc.png)

# Features:
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clean, responsive interface
- Lightweight — no external libraries or frameworks

# Built With:
- **HTML** – structure
- **CSS** – styling

# Getting Started
# Run it locally

1. Clone the repository

   git clone https://github.com/Mshibe-dotcom/MyCalculator.git
  
2. Navigate into the project folder
   
   cd MyCalculator
   
3. Open `calculator.html` in your browser
   
   open calculator.html   # macOS
   start calculator.html  # Windows
